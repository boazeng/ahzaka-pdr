🔧 PDR-001: מערכת דיווח חברת אחזקה

בפיתוח

📋 סקירה כללית

מערכת דיווח מקיפה לחברת אחזקה המאפשרת לטכנאים, מנהלים ולקוחות לדווח ולעקוב אחר תקלות, שעות עבודה ומשימות שבוצעו. המערכת פועלת דרך WhatsApp/Telegram ושולחת סיכומים במייל.

👥 משתמשי המערכת

🔨 טכנאים בשטח

  • דיווח על הגעה/יציאה מאתר
  • דיווח תקלות + צילום תמונות
  • עדכון סטטוס משימה
  • דיווח שעות עבודה

👔 מנהלים

  • קבלת סיכומים יומיים/שבועיים
  • צפייה בכל הדיווחים
  • הקצאת משימות לטכנאים
  • אישור שעות עבודה

🏢 לקוחות

  • פתיחת קריאת שירות
  • מעקב אחר סטטוס תקלה
  • קבלת עדכונים אוטומטיים
  • אישור סיום עבודה

⚙️ פיצ'רים עיקריים

  • דיווח תקלות - תיאור, רמת דחיפות, קטגוריה
  • צילום תמונות - לפני/אחרי, תיעוד תקלה
  • דיווח שעות - כניסה, יציאה, הפסקות
  • מעקב משימות - פתוח, בטיפול, הושלם
  • מיקום GPS - אימות נוכחות באתר
  • סיכום במייל - דוחות אוטומטיים יומיים

📱 ערוצי תקשורת

  • WhatsApp Bot - ממשק ראשי לדיווחים
  • Telegram Bot - חלופה ל-WhatsApp
  • Email - קבלת סיכומים ודוחות

🔄 זרימת עבודה

טכנאי מדווח ב-WhatsApp
↓
Bot מעבד את ההודעה + תמונות
↓
שמירה בבסיס נתונים
↓
עדכון למנהל + ללקוח
↓
סיכום יומי במייל

🛠️ ארכיטקטורה טכנית

  • WhatsApp API - Twilio / WhatsApp Business API
  • Telegram API - Bot API
  • Backend - AWS Lambda / Python
  • Database - DynamoDB / PostgreSQL
  • Storage - S3 לתמונות
  • Email - SES / SendGrid

📅 שלבי פיתוח

  • שלב 1: Bot בסיסי - דיווח תקלות + תמונות
  • שלב 2: דיווח שעות עבודה
  • שלב 3: ממשק מנהלים
  • שלב 4: התראות ללקוחות
  • שלב 5: דוחות אוטומטיים במייל
